# AIZero Core Module

AIZero forms the base AI engine of the RefreshCode ecosystem.

## Files
- Main.py: Core runtime controller.  
- ScikitLearn.py: Model training and evaluation using scikit-learn.  
- requirements.txt: Dependencies for ML operations.

## Purpose
Provides a lightweight experimental environment for training machine learning models and testing data-driven automation logic.

## Setup
1. Create a virtual environment.  
2. Run `pip install -r requirements.txt`.  
3. Start the module with `python Main.py`.
