# Lengtokens - Token Tracking and Cost Management

Python utility for monitoring token usage and estimating costs in AI models.

## Files
- calculode_precios.py: Cost calculation logic.  
- funcion.py: Helper utilities.  
- tokens_saldo_tracker.py: Token usage tracker.  
- saldo_tokens.txt: Token summary.  
- logs/: Session and cost history.

## Purpose
Designed to monitor OpenAI and Azure token usage for AI workflows, providing transparency and budgeting support.
