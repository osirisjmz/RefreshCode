# VisionAI - Image and Face Analysis

Python-based toolkit for computer vision using Azure Cognitive Services.

## Files
- Vision.py: Orchestrator for visual tasks.  
- ClasificacionDeImagenes.py: Image classification module.  
- DetectionFaceClient.py: Face detection and recognition handler.  
- bibliotecas.py: Shared library imports and setup.

## Purpose
Provides visual intelligence features such as classification, tagging, and facial recognition for automation and analytics systems.
