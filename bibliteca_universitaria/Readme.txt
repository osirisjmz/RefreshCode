# Biblioteca Universitaria

Flask-based web application for managing academic library operations.

## Files
- app.py: Main application file.  
- db_connection.py: Database connection logic.  
- requirements.txt: Dependencies.  
- static/: CSS and JS assets.  
- templates/: HTML pages (catalog, users, loans, reports, etc.).

## Purpose
Supports library administration, including catalog management, user registration, and tracking of loans and penalties.
