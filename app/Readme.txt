# Academic Management System

A Python Flask web application for managing academic entities such as students, teachers, courses, and grades.

## Structure
app/
├── appmain.py  
├── alumnos/  
├── calificaciones/  
├── cursos/  
├── maestros/  
├── materias/  
└── Templates/

Each section contains:
- alta.html: Create record.  
- baja.html: Delete record.  
- modificar.html: Update record.

## Purpose
Provides CRUD operations for an educational environment, designed to connect with databases and Power Automate processes.
