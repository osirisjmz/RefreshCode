# Python Practices Collection

Educational repository containing Python fundamentals examples.

## Categories
### Variables and Types
- variables.py, typevariables.py, typefunction.py  
- outputvariables.py: Output and formatting.

### Operators and Logic
- Operators.py: Core arithmetic and logical operators.  
- unpack_collection.py: Tuple unpacking and assignment.

### Strings and Text
- pythonstrings.py, StringMethods.py, StringFormat.py  
- slicing.py, SliceToTheEnd.py: Substring manipulation.  
- RemoveWhitespace.py: Data cleaning.

### Collections
- StringArrays.py, stringsarrays.py: List and string arrays.

## Purpose
Serves as a practice suite for learning Python syntax, strings, types, and operations.
